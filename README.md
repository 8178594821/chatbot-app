# Chatbot App 🤖

This is my AI chatbot project built using assistant-ui.

## 🚀 Getting Started

First, add your OpenAI API key to `.env.local` file:

OPENAI_API_KEY=your_api_key_here

## ▶️ Run the project

npm run dev

Then open:
http://localhost:3000

## 🛠️ Tech Used
- Next.js
- assistant-ui
- OpenAI API

## 📌 About
This project is created to build and test an AI chatbot.
